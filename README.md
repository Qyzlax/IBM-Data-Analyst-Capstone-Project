# IBM-Data-Analyst-Capstone-Project
This contains the files and presentation for my capstone project for the IBM Data Analyst Professional Certificate.

## Overview

This repository contains my completed capstone project for the **IBM Data Analyst Professional Certificate** offered through Coursera.

The project demonstrates the end-to-end data analysis workflow by applying the skills developed throughout the certification program. Using real-world developer survey data, the project covers every stage of the analytics lifecycle, from data collection and cleaning to exploratory analysis, visualization, dashboard creation, and presenting actionable insights.

This capstone showcases practical experience with Python, SQL, data visualization, and business intelligence tools in a real-world business scenario.

---

## Project Objectives

The primary objectives of this project were to:

- Collect data from multiple sources
- Clean and prepare data for analysis
- Perform exploratory data analysis (EDA)
- Identify trends in developer technologies and careers
- Create meaningful visualizations
- Build an interactive dashboard
- Present findings through a professional presentation

## Project Workflow

### 1. Data Collection

Collected developer survey data using APIs, web scraping, and provided datasets.

### 2. Data Wrangling

- Removed duplicate records
- Handled missing values
- Normalized data
- Prepared datasets for analysis

### 3. Exploratory Data Analysis

Performed statistical exploration to identify:

- Data distributions
- Correlations
- Technology adoption
- Salary trends
- Developer demographics

### 4. Data Visualization

Created charts including:

- Histograms
- Bar Charts
- Pie Charts
- Scatter Plots
- Box Plots
- Stacked Charts

to communicate key insights effectively.

### 5. Dashboard Development

Built an interactive dashboard using IBM Cognos Analytics to visualize:

- Current Technology Usage
- Future Technology Trends
- Developer Demographics

### 6. Final Presentation

Summarized findings into a professional presentation highlighting key insights and recommendations.
